Subnetting Network

Subnetting is a network technique used to divide a large IP network into smaller, more manageable segments called subnets. It helps improve network performance, security, and efficient IP address usage.

In subnetting, the IP address is split into a network portion and a host portion by using a subnet mask, which determines how many devices can exist within each subnet.

By creating multiple smaller subnets, organizations can reduce broadcast traffic, isolate different departments, and organize their networks better.
