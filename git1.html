Git - Version Control System

Git is a distributed version control system used to track changes in code during software development. It allows developers to work together, manage versions, and maintain the history of every update.
Git helps teams collaborate effectively by using like commits, branches, merges, repositories, and remote servers.

Git was created by Linus Torvalds in 2005 and is widely used across modern development teams. It is powerful, fast, and helps maintain a clean workflow for developers.
